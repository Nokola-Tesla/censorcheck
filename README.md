# Censorcheck

Набросал небольшой скрипт-чекер для проверки блокировок и цензуры на сервере, будет полезен для VPS/VDS с локацией РФ.
Скрипт анализирует, заблокирован ли тестируемый сервис на разных уровнях (DNS, IP, HTTP/HTTPS, DPI, REGIONAL), и выводит результаты в таблице.

Добавлены списки веб-сайтов, которые часто блокируются DPI и имеют географические ограничения. В дальнейшем список будет обновляться и дополняться.

Для каждого домена определяется статус:
OK — доступен
BLOCKED — заблокирован (указывается тип блокировки)
PARTIAL — частично доступен

Устанавливает недостающие зависимости (curl, nslookup, nc и netcat), если они отсутствуют.

Стоит учитывать, что точность определения блокировок составляет +- 90%, поэтому при необходимости рекомендуется делать ручную перепроверку.
Пожелания, предложения в ЛС.

Команда для запуска:
wget -qO- censorcheck.tlab.pw | bash
